# Texloren Apparel Sourcing - Landing Page

This is a modern, responsive landing page built for Texloren Apparel Sourcing, a Bangladesh-based OEM & private label apparel manufacturing company.

---

## Live Website

https://muzamal478.github.io/texloren-apparel/

---

## About This Project

This website is designed to showcase apparel manufacturing services, product categories, company information, testimonials, FAQs, and contact options in a clean and professional layout.

---

## Features

- Responsive design for mobile, tablet, and desktop
- Modern dark UI design
- Hero section with call-to-action buttons
- About section
- Services section
- Product categories section
- Stats section
- Testimonials section
- FAQ section
- Contact form UI
- WhatsApp floating button
- Back to top button
- Smooth scrolling and animations
- SEO friendly structure
